# Portfolio Website

## Installation

1. Install dependencies:
```bash
npm install
```

## Development

Run the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
